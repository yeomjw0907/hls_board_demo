# 게시판(프로토타입)



## 설치 및 실행

```bash
npm install
npm run dev
```

## 테스트 계정

로그인 페이지에서 다음 계정으로 테스트할 수 있습니다:

- `demo1@example.com` / `1234` 
- `demo2@example.com` / `1234` 
- `demo3@example.com` / `1234` 
- `demo4@example.com` / `1234` 
- `admin@example.com` / `admin` 

## 주요 기능

- 로그인/로그아웃
- 게시글 작성 (carrier_user, super_admin)
- 게시글 목록 조회, 정렬, 필터
- 거래 의도 선언 (댓글 작성)
- 거래 수락/거절
- Super Admin 게시글 편집/삭제
